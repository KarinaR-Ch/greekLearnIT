 IT-лексика для тестувальника: українсько-грецький словник
Інтерактивний додаток для вивчення IT-лексики українсько-грецькою мовами, спеціально розроблений для тестувальників JavaScript.

Особливості
- 100 професійних термінів для тестувальників JavaScript
- Інтерактивні картки для швидкого запам'ятовування
- Автоматичне оновлення - при вивченні картки додається нова
- Система прогресу з використанням LocalStorage
- Фільтрація за категоріями та статусом вивчення
- Адаптивний дизайн для всіх пристроїв
- Привабливий UI з сучасними анімаціями

Структура проєкту
Технології
- HTML5 - семантична розмітка
- CSS3 - сучасні стилі з анімаціями
- Vanilla JavaScript - чистий JS без бібліотек
- LocalStorage API - збереження прогресу
- Font Awesome - іконки
- CSS Grid & Flexbox - верстка

Як користуватися
1. Переглядайте картки - клікайте на картку, щоб перевернути її
2. Оцінюйте свої знання:
   - "Знаю" - картка зникає, додається нова
   - "Повторити" - картка залишається для подальшого вивчення
3. Фільтруйте картки:
   - Усі слова
   - Вивчені
   - Потрібно повторити
   - Загальні терміни
4. Використовуйте функції:
   - Перемішати картки
   - Показати ще 10 карток
   - Скинути прогрес

Функціональність
- Створення карток - динамічне генерування інтерактивних карток
- Управління прогрессом - автоматичне збереження в LocalStorage
- Анімації - плавні переходи, перевертання, зникнення карток
- Адаптивність - повна підтримка мобільних пристроїв
- Доступність - keyboard navigation та ARIA attributes
